# VX-9 Mod for Minecraft Fabric

A comprehensive Minecraft 1.20.1 Fabric mod featuring the VX-9 laser weapon system, quantum crafting mechanics, skill trees, and custom biomes.

## Features

### 🔫 VX-9 Laser Gun
- 6 progressive levels
- Experience-based progression
- Level 6 requires Powered Quantum Core
- Multi-laser attack at higher levels

### ⚙️ Quantum Crafting System
- **Quantum Core Charger**: Charges quantum cores
- **VX-9 Crafting Station**: 8x8 grid crafting interface
- **Crafting Pedestal**: Crafting display block

### 🏔️ Custom Biomes
1. **Quantum Nexus** - High-tech quantum landscape
2. **Skill Sanctum** - Training grounds for skill advancement
3. **Boss Citadel** - Dangerous boss arena
4. **Fabrication Core** - Fabricium ore deposits

### 💎 Metals & Resources
- **Vx9ium** - Primary quantum metal
- **Nexium** - Advanced processing material
- **Chronium** - Time-related properties
- **Fabricium** - Fabrication material

### 🎮 Skill Tree System
- **Tier 1 (Root)**: Laser Mastery, Quantum Affinity, Combat Training
- **Tier 2 (Advanced)**: Laser Precision, Quantum Surge, Weapon Mastery
- **Tier 3 (Ultimate)**: Perfect Shot, Quantum Realm

### 👹 Enemies & Bosses
- **Quantum Boss**: Ultimate quantum enemy (250 exp)
- **Crono Boss**: Time-manipulation boss (300 exp)
- **Fabrication Mob**: Standard hostile mob (50 exp)

### 🛡️ Armor & Weapons
- Quantum armor set
- Nexium Sword
- Chronium Staff
- Fabricium Axe

## Installation

1. Download Fabric Loader 0.14.22+ for Minecraft 1.20.1
2. Install Fabric API
3. Place mod jar in `mods` folder
4. Launch the game

## Crafting Recipes

### Powered Quantum Core
```
N N N
N Q N
N N N
```
(N = Netherite Block, Q = Quantum Core)

### VX-9 Laser Gun (Level 1)
Requires crafting at the VX-9 Crafting Station
- 4 Netherite Blocks
- 4 Diamonds
- 1 Powered Quantum Core
- 4 Nether Stars

## Configuration

Edit `config/vx9mod.properties` to customize:
- Laser damage values
- Quantum core charge rates
- Biome generation rates
- Mob spawn rates
